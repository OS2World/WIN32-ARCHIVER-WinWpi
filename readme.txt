WinWpi
------
(c) 2001-2002 Cornelis Bockemuehl

Command line WarpIN (WPI) archive lister / extractor running on a Win32
system (or probably also DOS with support for long filenames). Just start
the program from a command line with no parameters and it explains itself!

If you have any further questions or problems: cbockem@tiscalinet.ch
